Data Structures In C++

CSCI-1200 is RPI's Data Structures in C++ course required for all Computer Science and Information Technology and Web Science Majors. The course description is as follows:

This course covers fundamental data structures and their use in programming. This includes both class design and features of the C++ programming language. Much of our discussion will be built around the design and use of the C++ standard library (STL). By using the standard library, students will be able to write reasonably sophisticated programs quickly.

Unfortunately due to this course's policy, I am unable to upload or publish any of my work until I graduate. Coding samples can be provided upon request.

Computer Science I

CSCI-1100 is RPI's introductory Computer Science course required for all Computer Science and Information Technology and Web Science Majors. The course description is as follows:

An introduction to computer programming, algorithm design, and analysis. Additional topics include basic computer organization; internal representation of scalar and array data; use of top-down design and subprograms to tackle complex problems; abstract data types. Enrichment material as time allows. Interdisciplinary case studies, numerical and non-numerical applications.

Roche-Genia Lab Internal Lab Site

This project was completed during a summer internship with Genia Technologies, a division of Roche Sequencing. This site was built with the intention of consolidating lab manuals and giving people the ability to perform common lab calculations online. This not only made the process simpler and faster and provided a centralized location for lab manual storage.

Due to the sensitive nature of the lab manuals and information stored in the site, a sample will only be provided on request.
